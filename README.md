## Training Rasa models
```shell
rasa train
```

## Start Rasa action server
```shell
rasa run actions
```

## Start Rasa server and client
```bash
rasa shell
```

Try to type some queries like `raise your left hand 40 degrees for 3 seconds` and see the response.

Have Fun!
